# Cu Artificial "Inteligence"

A simple bot project implementing the [CuGS](https://github.com/gfpacheco/cugs) Player API.

## AIs

### `dumb`

Never plays any card, always pass.

### `play-first`

Plays the first set of cards found in hand.

## Running

- Setup (once):
  - `npm install`
- Run server:
  - `[PORT=3000] npm start`
